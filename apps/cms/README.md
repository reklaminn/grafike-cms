# CMS App

Bu klasör hedefte Laravel tabanlı admin/CMS uygulamasını barındıracak.

Sorumluluklar:

- kullanıcı / rol / yetki
- sayfa / makale / medya / SEO / form yönetimi
- theme pack metadata
- page templates
- site instances
- public API

İlk taşınacak parçalar:

- mevcut admin panel
- mevcut içerik modeli
- mevcut SEO / redirect mantığı
- mevcut medya yönetimi

Sonradan eklenecekler:

- template kayıtları
- section template registry yönetimi
- theme token paneli
- global custom css/js
- siteyi template olarak kaydetme
